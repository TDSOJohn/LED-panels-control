## DMX LED driver

Use any HUB12 display with DMX control.

### Dependencies

- [DMD2](https://github.com/freetronics/DMD2)
- [DMXSerial](https://github.com/mathertel/DmxSerial)

### DMX Control

| Channel | Function |
| ------- | -------- |
| 1 | speed (slow to high) |
| 2 | pattern |
| 3 | reset / control |
